#### 文件寻找方法
```
【Windows】
# 全盘查找文件，一定要加一个星号！
for /r c:\ %i in (password.txt*) do @echo %i
for /r c:\ %i in (*.ini) do @echo %i

# 查找 C 盘中包含 password 字样的文件，一定要双引号！
findstr /s /n "password" c:\*

# 查找 pwd.txt 中是否包含 password 字样，一定要双引号！
find /N /I "password" pwd.txt

【Linux】
# 查找文件
find / -name filename.ext

# 全盘查找含有 flag 的文件
grep flag -r /
```
