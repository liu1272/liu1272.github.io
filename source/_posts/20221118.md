### 每日一记
#### 面向对象编程
类:设计
对象:实例

#### 类的命名
驼峰命名法

#### 语法
```
class 类名:
    属性(变量) = 值
    def 函数名(self):         //方法(函数),一定要写self
        执行代码
对象名=类名()                 //创建对象
对象名.函数()                 //调用对象
```

#### 魔法方法
通常格式为__方法名__
```
def __init__(self,值):
    self.属性=值

如果值也是一个变量就能随时传参进行更改
```

#### 过程
1.定义类class
2.创建内存空间,定义属性和方法
3.实例化类,创建对象
4.调用类的方法
